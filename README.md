# DanaZone™ — World-Class E-Commerce Website

Kenya's premier online marketplace. Built to Google Merchant Centre (GMC) standards.

## File Structure
```
danazone/
├── index.html              ← Homepage
├── sitemap.xml             ← Google sitemap
├── robots.txt              ← Search engine rules
├── css/
│   └── style.css           ← All styles (Deep Navy + Teal + Gold)
├── js/
│   ├── components.js       ← Header, Footer, Cart logic
│   └── products.js         ← Product data + render helpers
└── pages/
    ├── products.html       ← Product listing with filters
    ├── product.html        ← Single product detail
    ├── deals.html          ← Flash deals page
    ├── checkout.html       ← Checkout (M-Pesa, PayPal, etc.)
    ├── account.html        ← Customer account / orders
    ├── contact.html        ← Help centre + contact form
    ├── about.html          ← About DanaZone
    ├── seller.html         ← Sell on DanaZone
    ├── track-order.html    ← Order tracking
    ├── shipping.html       ← Shipping policy (GMC required)
    ├── returns.html        ← Returns/refund policy (GMC required)
    ├── privacy.html        ← Privacy policy (GMC required)
    └── terms.html          ← Terms & conditions (GMC required)
```

## Color Scheme
- **Primary:** Deep Navy `#0A0E1A`
- **Accent:** Teal `#00BFA6`
- **Highlight:** Gold `#F5C842`
- **Background:** Off-white `#F8FAFB`

## Fonts
- **Headings:** Syne (Google Fonts)
- **Body:** DM Sans (Google Fonts)

## GMC Requirements Included
- ✅ Privacy Policy page
- ✅ Shipping Policy page (with exact rates + times)
- ✅ Returns/Refund Policy page
- ✅ Terms & Conditions page
- ✅ Contact page with address
- ✅ HTTPS-ready (add SSL on your host)
- ✅ Structured Data (Schema.org) for WebSite + Organization
- ✅ Sitemap.xml
- ✅ robots.txt
- ✅ Product pages with prices, descriptions, availability

## To Deploy on GitHub Pages
1. Push to https://github.com/KigHeart/danzoneWeb
2. Go to Settings → Pages → Deploy from main branch
3. Your site will be live at https://kigheart.github.io/danzoneWeb/

## Next Steps for GMC
1. Connect to Google Merchant Centre at merchants.google.com
2. Verify ownership via Google Search Console
3. Create a product feed (manual Google Sheet or auto via API)
4. Submit products and wait for approval
